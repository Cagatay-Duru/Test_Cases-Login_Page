 Minimal Login Page

A simple, purely black & white login page built without any external libraries or frameworks. This project was created to implement and test standard login test cases and UI validations.

 Features
- Form Validation: Checks for empty username and password fields before submission.
- Boundary Value Checking: Password length is restricted strictly to 8-12 characters.
- Basic SQL Injection Prevention**: Built-in logic to detect and block common SQL injection patterns (e.g., `' OR '1'='1`).
- Mock Dashboard: On successful login with valid credentials, it redirects to a simple dummy dashboard.
